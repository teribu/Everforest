# 🌿 Everforest

A Roblox UI library inspired by the osu! interface, built with Luau.

## ⚡ Features

* osu!-inspired visual design
* Theme support
* Modular architecture
* Lightweight and customizable
* Single-file release builds

## 📦 Installation

```lua
local Everforest = loadstring(game:HttpGet(
    "https://github.com/teribu/Everforest/releases/latest/download/main.lua"
))()
```
